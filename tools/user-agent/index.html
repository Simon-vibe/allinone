User Agent FAQ

A User Agent is a string of text that your browser sends to websites to identify itself.

Why do need this?

  • Troubleshooting: Support teams often ask for this to diagnose issues.
  • Content Delivery: Websites use it to serve mobile or desktop versions.
  • Analytics: Helps webmasters understand their audience's devices.

How to change my User Agent?

You can use developer tools in your browser or extensions to 'spoof' or change your User Agent string.

Is modifying User Agent safe?

Yes, it generally is safe, but some websites might look broken if they rely on it for feature detection.
