User Agent FAQ

A User Agent is a string of text that your browser sends to websites to identify itself.

Why do need this?

  • Troubleshooting: Support teams often ask for this to diagnose issues.
  • Content Delivery: Websites use it to serve mobile or desktop versions.
  • Analytics: Helps webmasters understand their audience's devices.
